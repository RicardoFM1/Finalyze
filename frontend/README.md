# Finalyze Frontend

## Setup
1. Install dependencies:
   ```sh
   npm install
   ```
2. Run development server:
   ```sh
   npm run dev
   ```

## Structure
- **Views**: Located in `src/views`. Contains pages like Home, Plans, Login, Dashboard.
- **Components**: Located in `src/components`. Reusable UI parts.
- **Router**: Configured in `src/router/index.js`.
- **Plugins**: Vuetify and other plugins in `src/plugins`.

## Key Features
- **Authentication**: Login/Register views ready.
- **Plans**: Validation of subscription plans.
- **Dashboard**: Financial overview.
- **Payment**: Mercado Pago Brick integration skeleton.
